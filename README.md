# Portfolio Website

My personal portfolio website built with GitProfile.

**Based on:** [arifszn/gitprofile](https://github.com/arifszn/gitprofile)

## What I Changed

- Updated `gitprofile.config.ts` with my information:
  - GitHub username
  - Social media links
  - Research experience and education
  - Publications and projects
  - Skills and certifications
- Customized theme and styling preferences
- Configured deployment settings for GitHub Pages
